# Netflix-Style Movie Recommendation Engine

## Project Overview
This project builds a movie recommendation engine using Python, Pandas, Scikit-Learn, and cosine similarity.

The goal is to simulate how streaming platforms recommend similar content based on user rating behavior.

## Business Problem
Streaming platforms like Netflix rely on personalized recommendations to increase user engagement, viewing time, and retention.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook
- Cosine Similarity

## Dataset
MovieLens ml-latest-small dataset from GroupLens.

## Project Steps
1. Loaded movie and rating data
2. Merged movie metadata with user ratings
3. Built a user-movie rating matrix
4. Calculated movie similarity using cosine similarity
5. Created a recommendation function
6. Generated sample recommendations

## Sample Use Case
Input movie:

`Matrix, The (1999)`

Output:

Top 10 similar movie recommendations based on user rating patterns.
